## General information

This module supports listing and extracting content from Ren'Py visual novel engine archives (RPA).
Supported RPA file format versions:
    - 3.0

## How to install

1. Put all files to %FARHOME%\Plugins\Observer\modules\
2. Add "RenPy=modules\renpy.so" to "[Modules]" section of observer.ini
3. Add "RenPy=*.rpa" to "[Filters]" section of observer.ini
